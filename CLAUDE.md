# プロジェクト設定

## コミュニケーション
- **言語**: 日本語
- このプロジェクトでの全てのコミュニケーションは日本語で行ってください

## プロジェクト概要
- **目的**: 「datamanagement」という本のためのコード管理
- **主要言語**: Rust
- **対象領域**: 画像処理と圧縮技術の比較検討

## 技術要件

### 画像フォーマット
- JPEG (.jpg)
- PNG (.png) 
- WebP (.webp)

### 圧縮フォーマット
- 7-zip
- Zstandard (zstd)
- 圧縮効率の比較検討を行う

## 開発環境
- **言語**: Rust
- **プラットフォーム**: Linux (WSL2)

## タスク管理
- 画像処理ライブラリの選定
- 圧縮ライブラリの統合
- パフォーマンス比較の実装
- ベンチマーク結果の分析

## 注意事項
- 全てのコード、コメント、ドキュメントは日本語で記述
- 実装時はRustのベストプラクティスに従う
- 圧縮効率とパフォーマンスの両方を考慮した実装を行う
